# Apartmani Ante

Modern SvelteKit website for a private apartment rental business in Makarska, Croatia.

## Stack

- SvelteKit
- Tailwind CSS + LESS
- FullCalendar
- Neon Postgres
- Nodemailer

## Features

- Public apartment listing and apartment detail pages
- Availability calendar and reservation request flow
- Private owner dashboard for bookings and requests
- SEO setup with sitemap, robots and structured data

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
